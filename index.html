DHCP Explained for CCNA

Your guide to understanding and configuring DHCP

Theory

What is DHCP?

DHCP (Dynamic Host Configuration Protocol) is a network protocol that automates the assignment of IP addresses and other network settings to devices in a network.

How DHCP Works (DORA)

  • Discover: Client broadcasts a request for a DHCP server.
  • Offer: Server offers an IP address.
  • Request: Client requests the offered IP address.
  • Acknowledge: Server confirms the IP assignment.

Practical Configuration

Configuring DHCP on a Cisco Router


Router> enable
Router# configure terminal
Router(config)# ip dhcp pool POOL_NAME
Router(dhcp-config)# network 192.168.1.0 255.255.255.0
Router(dhcp-config)# default-router 192.168.1.1
Router(dhcp-config)# dns-server 8.8.8.8
Router(dhcp-config)# lease 7

Excluding IP Addresses


Router(config)# ip dhcp excluded-address 192.168.1.1 192.168.1.10
