Partie Théorique

1. Définition et Rôle de DHCP

Le DHCP est un protocole réseau qui automatise l'attribution des adresses IP et d'autres paramètres réseau aux appareils d'un réseau.

But principal : Simplifier la configuration des appareils sur un réseau, éviter les conflits d'adresses IP et réduire les erreurs humaines.

Paramètres configurés par DHCP :

  • Adresse IP
  • Masque de sous-réseau
  • Passerelle par défaut
  • Serveurs DNS
  • Durée de bail

2. Processus DORA

1. Discover (Découverte)

Le client envoie un message de diffusion pour rechercher un serveur DHCP.

2. Offer (Offre)

Le serveur DHCP répond avec une offre contenant une adresse IP disponible.

3. Request (Demande)

Le client répond au serveur pour accepter l'offre.

4. Acknowledge (Accusé)

Le serveur confirme l'attribution de l'adresse IP.

Partie Pratique

Configuration d'un Serveur DHCP sur un Routeur Cisco

Important : Assurez-vous d'être en mode privilégié avant de commencer la configuration.

Étape 1 : Entrer en mode configuration

Router> enable
Router# configure terminal

Étape 2 : Créer un pool DHCP

Router(config)# ip dhcp pool NOM_DU_POOL
Router(dhcp-config)# network 192.168.1.0 255.255.255.0
Router(dhcp-config)# default-router 192.168.1.1
Router(dhcp-config)# dns-server 8.8.8.8
Router(dhcp-config)# lease 7
Chaque commande configure un aspect spécifique du service DHCP

Exercice Pratique

Objectif

Configurer un serveur DHCP sur un routeur pour un réseau 192.168.10.0/24 avec :

  • Plage DHCP : 192.168.10.11 à 192.168.10.254
  • Passerelle par défaut : 192.168.10.1
  • Serveur DNS : 8.8.8.8
  • Durée de bail : 3 jours

Correction de l'Exercice

Objectif : Configuration complète d'un serveur DHCP selon les spécifications données.

Étapes de la solution :

1. Accéder au mode de configuration

Router> enable
Router# configure terminal

2. Exclure les adresses réservées

Router(config)# ip dhcp excluded-address 192.168.10.1 192.168.10.10
Cette commande réserve les 10 premières adresses pour les équipements statiques

3. Créer et configurer le pool DHCP

Router(config)# ip dhcp pool RESEAU_LOCAL
Router(dhcp-config)# network 192.168.10.0 255.255.255.0
Router(dhcp-config)# default-router 192.168.10.1
Router(dhcp-config)# dns-server 8.8.8.8
Router(dhcp-config)# lease 3

4. Vérification de la configuration

Router# show running-config | section dhcp
Router# show ip dhcp binding
Router# show ip dhcp pool

Validation

Pour vérifier que votre configuration fonctionne :

  1. Connectez un PC au réseau
  2. Configurez l'interface réseau en DHCP
  3. Vérifiez que le PC reçoit une adresse IP dans la plage définie
  4. Testez la connectivité avec un ping vers la passerelle
